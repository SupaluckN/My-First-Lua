# My-First-Lua

## How to run Lua.
I use lua54.
Use this command : 

``` 
lua54 ./main.lua 
```

or 

``` 
lua54 main.lua 
```

## How to comment in Lua 
### Mutiline comments in Lua. 
```
--[[
This is multiline comments in Lua. 
]]--
```

### One line comment in Lua. 
```
--This is a one line comment in Lua.
```

or 

```
--This is a one line comment in Lua.--
```

## How to Print or Output on the screen in Lua
